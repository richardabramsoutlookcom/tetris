# Requirements: Tetris Game

**Defined:** 2026-01-17
**Core Value:** Classic Tetris gameplay that feels good to play — responsive controls, satisfying feedback, and the option to experience it in authentic Gameboy style.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### High Scores

- [ ] **SCORE-01**: User can see their final score on a top 10 leaderboard
- [ ] **SCORE-02**: User can enter initials/name when achieving a high score
- [ ] **SCORE-03**: User's high scores persist across browser sessions (localStorage)

### Audio

- [ ] **AUDIO-01**: User can hear Type A (Korobeiniki) music during gameplay
- [ ] **AUDIO-02**: User can hear Type B music during gameplay
- [ ] **AUDIO-03**: User can hear Type C music during gameplay
- [ ] **AUDIO-04**: User can select which music track plays (or none)

### Visual Themes

- [ ] **THEME-01**: User can play in Gameboy DMG mode (green-tinted retro aesthetic)
- [ ] **THEME-02**: User can toggle between modern and Gameboy visual modes

### Mobile

- [ ] **MOBIL-01**: User can play using on-screen touch controls on mobile devices
- [ ] **MOBIL-02**: Touch controls include D-pad for movement and buttons for rotate/hold/drop

## v2 Requirements

*None — all proposed features included in v1*

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Online multiplayer | Complexity exceeds project goals |
| Cloud sync/online leaderboards | Keeping it local-only for simplicity |
| Other retro console themes (NES, SNES) | Focusing on Gameboy only for v1 |
